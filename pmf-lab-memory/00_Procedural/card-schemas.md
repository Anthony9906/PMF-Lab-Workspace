# Card Schemas

## Opportunity Card

- id
- title
- manufacturing domain
- user or buyer
- pain statement
- current workaround
- agent fit
- value hypothesis
- feasibility hypothesis
- risk level
- evidence status
- score
- decision

## Experiment Card

- id
- linked opportunity id
- hypothesis
- test design
- success criteria
- data needed
- owner
- due date
- result
- next action

## Feedback Card

- id
- linked opportunity or experiment id
- observed outcome
- mismatch with expectation
- root cause
- lesson learned
- promote / revise / retire
