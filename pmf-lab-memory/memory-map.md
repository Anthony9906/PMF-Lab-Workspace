# PMF Lab Memory Map

This folder is the working memory for AI+manufacturing opportunity discovery.
It is optimized for human + agent co-work and for repeated learning over 3-6 months.

## Layers

### 00_Procedural

Stable workflow, guardrails, schemas, and operating rules.

### 10_Working

Active opportunity pool, current hypotheses, live experiments, and open questions.

### 20_Canonical

Validated patterns, reusable playbooks, approved definitions, and durable decisions.

### 30_Source

Raw evidence, interviews, documents, screenshots, logs, and external references.

### 40_Feedback

Outcome logs, evaluation results, failure reasons, and promotion/retirement notes.

## Operating rules

1. Keep source, judgment, and conclusion separate.
2. Move stable knowledge upward only after repeated confirmation.
3. Keep working memory short and current; archive aggressively.
4. Never overwrite source truth.
5. Human decides opportunity priority and PMF call.
6. Agent may draft, compare, cluster, score, and propose, but not silently finalize.

## Default flow

Capture -> Normalize -> Cluster -> Prioritize -> Hypothesize -> Experiment -> Evaluate -> Promote or retire
